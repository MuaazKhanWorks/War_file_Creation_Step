
1. Below <Parents> first Past '<packaging>war</packaging>'.

2. Include Below Dependancy:
		<dependency>
			<groupId>org.springframework.boot</groupId>
			<artifactId>spring-boot-starter-tomcat</artifactId>
			<scope>provided</scope>
		</dependency>
		
3. If u want .war to be as ur project name -> Past <finalName>${artifactId}</finalName> after <build>

4. From Main class extends SpringBootServletInitializer.

5. Include Below in Main Class:

	@Override
	protected SpringApplicationBuilder configure(SpringApplicationBuilder builder) {
		return builder.sources(RestTempleteApplication.class);
	}
	
6. Delete the Old target File.

7. Go to Mavin Reload place -> Click and open ur project Double Click 'Clean'. Then Double Click 'Package'.

8. .war File will be successfully generated inside target folder.
